# Gatsby Contentful Project Example

## Features

[Gatsby] project including

- [TypeScript]
- [MDX v2]
- [ESLint]
- [Prettier]
- [husky]
- [lint-staged]

## Quick start

```sh
$ git clone git@github.com:cieloazul310/gatsby-typescript-mdx-starter.git
$ yarn install

# husky install
$ yarn run prepare

# develop
$ yarn run start

# build
$ yarn run build

# serve
$ yarn run serve

# lint
$ yarn run eslint

# format
$ yarn run prettier
```

[TypeScript]: https://www.typescriptlang.org/
